## 📝 Description  
This project demonstrates **unit testing** in a restaurant management system using **JUnit 4 and JUnit 5**. It includes various test cases to validate key functionalities such as:  
✅ Order processing  
✅ Menu management  
✅ Payment validation  
✅ Table reservations  
✅ Inventory updates  

## 🚀 Technologies  
- Java  
- JUnit 4  
- JUnit 5
